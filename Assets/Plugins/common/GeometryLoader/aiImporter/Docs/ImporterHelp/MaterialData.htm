=MaterialData Class
aiImporter Runtime
 MaterialData Class

Represents a single material that contains color and texture information of the mesh.


Member Description
string MatName Name of the material.
string filename Filename of the material.
bool HasTextureDiffuse Checks if the material has a diffuse texture.
bool HasTextureNormal Checks if the material has a normal texture.
bool HasTransTexture Checks if the material has a transparency texture.
bool HasColorDiffuse Checks if the material has a color diffuse property.
bool HasColorEmissive Checks if the material has a color emissive property.
bool HasColorSpecular Checks if the material has a color specular property.
bool HasTransparency Checks if the material has a transparency value.
byte[] DiffuseTexture Byte array representation of the diffuse texture.*
byte[] NormalTexture Byte array representation of the normal texture.*
Vector4C DiffuseColor Gets the diffuse color.**
Vector4C SpecularColor Gets the specular color.**
Vector4C EmissiveColor Gets the emissive color.**
float AlphaValue Transparency value of the material. Between 0 and 1.



* Byte array will be converted to Texture2D format via Unity scripting.

** Vector4C is the serializable version of Vector4 class of Unity. Cast between these types are possible.
