=BoneData Class
aiImporter Runtime
 BoneData Class

Represents a single bone of a mesh.


Member Description
string BoneName Name of the bone. Can be used to find the bone in the hierarchy.
string ParentName Name of the parent bone, if there is any.
Vector3C position Position vector of the bone. *
QuaternionC rotation Rotation vector of the bone. *
Vector3C scale Scale vector of the bone. *
bool HasParent Checks if the bone has a parent bone.
Dictionary<string,AnimationData> animations All of the animation data that is related to this bone. Animation datas are stored separately for each animation. Animation name is used as Dictionary key.



* Vector3C and QuaternionC are serializable versions of Vector3 and Quaternion classes of Unity. Cast between these types are possible.
