=WeightData Class
aiImporter Runtime
 WeightData Class

Represents the influence that a bone has on a single vertex.


Member Description
int VertexID ID of the vertex that is being influenced by the bone.
int BoneID ID of the bone that is affecting the vertex.
float Weight Strength of the influence in a range of 0 to 1.
