=Model Class
aiImporter Runtime
 Model Class

Represents a single model


Member Description
string Name Name of the model node.
Vector3C position Position vector of the model. *
QuaternionC rotation Rotation values of the model in Quaternion format.(X, Y, Z ,W)*
Vector3C scale Scale vector of the model. *
List<Geometry> geometry List containing the geometry data of the model. (Meshes)



* Vector3C and QuaternionC are serializable versions of Vector3 and Quaternion classes of Unity. Cast between these types are possible./p>
