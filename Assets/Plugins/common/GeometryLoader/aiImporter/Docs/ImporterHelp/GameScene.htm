 GameScene Class

Represents a completely imported model or scene. Every data that aiImporter has collected so far can be accesed here. This is the scene that we pass to Unity to create a model at runtime.



This class consists mostly of Lists. We will use a for/for each loop in Unity to access the items of these lists.


Member Description
List<Model> models List containing all of the models that are imported.
List<MaterialData> materials List containing all of the materials that are used in the imported scene.
List<BoneData> bones List containing all of the bones that are used in the imported scene.
List<string> animNames List containing all of the animation names that are used by imported model.
