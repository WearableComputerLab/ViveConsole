aiImporter Runtime
 Features

Current Features

  • Loading mesh geometry.
  • Loading materials with textures.
  • Loading texture images that are normally not supported by Unity at runtime like .tga, .dds. (Thanks to DevIL library)
  • Loading separate transparency textures.
  • Loading skinning data. (Skeleton and bone weights)
  • Loading animations

Planned Features

  • Making aiImporter available for other platforms (Already in progress).

Supported Platforms

  • Windows
