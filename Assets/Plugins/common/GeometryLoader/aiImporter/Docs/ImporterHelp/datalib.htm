 DataLib Reference

This section contains the complete reference to DataLib library. The aim of the library is to create an object that enables the transfer of model data between the importer and Unity while still having the ability to be easily serialized/deserialized at the runtime.


The library does not contain any methods, just properties to hold the model data. Because of that it is pretty simple to understand and most of the time the property names are self-explanatory.
