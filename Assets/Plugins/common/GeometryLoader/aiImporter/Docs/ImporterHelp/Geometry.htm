=Geometry Class
aiImporter Runtime
 Geometry Class

Represents geometry of a model with a single material. (Mesh data)


Member Description
string Name Name of the mesh.
int MeshIndex Used by aiImporter to identify the Geometry class.
List<Vector3C> vertices List of vertex position vectors. *
List<Vector4C> vertexColors List of color values for each vertex. *
List<Vector3C> normals List of vertex normals. *
List<Vector2C> uvs List of uv positions. *
int[] triangles An array containing all triangles in the mesh.
int MaterialIndex Index of the material that is used by this mesh.
List<WeightDataList>weights The bone weights of each vertex.
List<BoneData> bones List of bones that are affecting this mesh.



* VectorXC and QuaternionC are serializable versions of VectorX and Quaternion classes of Unity. Cast between these types are possible./p>
